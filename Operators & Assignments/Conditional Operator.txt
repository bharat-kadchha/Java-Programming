        Java Operators.


    13. Conditional operator(?:) / ternary operator.
        -> In java it is only one ternary operator.
        -> nesting of Conditional operator is possible.
        
        Example 1:-

            int x = (10 < 20) ? 30 : 40;
            System.out.println(x); // 30


        Example 2:-

            int x = (10 > 20) ? 30 : (40 < 50) ? 60 : 70; 
            System.out.println(x); // 60
